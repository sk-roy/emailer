Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Send an email

Add an email to the queue for sending.

Example request:
const url = new URL(
    "http://localhost:8000/api/emails"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "subject": "\"Meeting Reminder\"",
    "email": "\"user@example.com\"",
    "message": "\"Don't forget our meeting tomorrow at 10 AM.\"",
    "attachment": "\"dGVzdCBhdHRhY2htZW50IGNvbnRlbnQ=\"",
    "attachment_filename": "\"file.pdf\""
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$url = 'http://localhost:8000/api/emails';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'subject' => '"Meeting Reminder"',
            'email' => '"user@example.com"',
            'message' => '"Don\'t forget our meeting tomorrow at 10 AM."',
            'attachment' => '"dGVzdCBhdHRhY2htZW50IGNvbnRlbnQ="',
            'attachment_filename' => '"file.pdf"',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Request      

POST api/emails

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

subject   string   

The subject of the email. Example: "Meeting Reminder"

email   string   

The recipient's email address. Example: "user@example.com"

message   string   

The body of the email. Example: "Don't forget our meeting tomorrow at 10 AM."

attachment   string  optional  

optional A base64 encoded string of the email attachment. Example: "dGVzdCBhdHRhY2htZW50IGNvbnRlbnQ="

attachment_filename   string  optional  

required_with:attachment The filename of the attachment. Example: "file.pdf"

Get email list

Retrieve a paginated list of emails.

Example request:
const url = new URL(
    "http://localhost:8000/api/emails"
);

const params = {
    "per_page": "15",
    "page": "18",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$url = 'http://localhost:8000/api/emails';
$response = $client->get(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'query' => [
            'per_page' => '15',
            'page' => '18',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
